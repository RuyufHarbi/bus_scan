# 🚌 Bus Scan

A mobile application developed as a **graduation project** to streamline and modernize student bus attendance and verification processes using QR code technology.

The system enables students to scan their QR codes when boarding the bus, providing an efficient, secure, and paperless method for attendance tracking.

---

## 📖 Project Overview

**Bus Scan** was developed as part of the requirements for obtaining a Bachelor's degree in Computer Science. The project aims to improve traditional bus attendance systems by introducing a digital solution that enhances efficiency and reduces manual errors.

### Key Features

- 📱 QR code-based student verification
- 🚌 Fast and secure bus attendance tracking
- 🔍 Real-time scanning process
- 📊 Improved accuracy compared to manual attendance methods
- ⚡ User-friendly mobile interface

## ⚙️ Local Deployment & Environment Setup

Follow the steps below to clone, configure, and run the application locally.

### Prerequisites

Before running the project, ensure you have the following installed:

- Flutter SDK (**Stable Channel**)
- Android Studio or Visual Studio Code
- Flutter Extension
- Dart Extension
- A testing environment:
  - Physical Android device with USB Debugging enabled, or
  - Android/iOS Emulator

---

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/RuyufHarbi/bus_scan.git
cd bus_scan
```

### 2. Install Dependencies

Download and synchronize all required project packages:

```bash
flutter pub get
```

### 3. Run the Application

Launch the application on your connected device or emulator:

```bash
flutter run
```
